# cyberwarrior-llm-challenge
This repository is dedicated to our project on LLM-based network attack detection and response for the CyberWarrior 2025 LLM Challenge.

## Installed software
The data parsing stage uses pyshark, which uses t-shark under the hood. Please refer to [this documentation](https://tshark.dev/setup/install/#install-wireshark-with-a-package-manager) to install t-shark for your system.
